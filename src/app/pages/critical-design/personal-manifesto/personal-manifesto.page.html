Assignment

Look for famous and less famous manifestos. Create a personal manifesto. Write a set of statements, rules or a more essay-like piece about your opinion on the worlds's big theme's: economics, politics, culture, society, esthetics etc. Try to formulate them very specifically. Design the manifesto in a way that suits it's content.

Approach

To create a personal manifesto, I first needed to know which values and themes should be included in it. From the critical problems I can and will make values that are important to me after which I will find a suitable form for it to be personal to me.

Execution

I created a list of values to be included inside of the personal manifesto. Through brainstorming with Mike Hendriks, I settled down for a form of the manifesto. It would become a manifest.json file in which I put my values inside of. The end result is great and very personal to me. It took some time to build the manifest itself, but it payed of in the end I think. The feedback received from the class was very positive and had no points of improvement.

Reflection

I found this assignment to be eye opening of some sorts. I never thought about actual values that I adhere to. So this is confronting at a level which I reflect on how I interact with the world.

Responsibility

I created the personal manifesto myself. Thanks Mike Hendriks for the cool manifest.json idea!
